# Golang 101

A small study repo for learning Go basics and concurrency (goroutines, channels, etc.).

## Requirements

- [Go](https://go.dev/) 1.21+ (or your installed version)

## Run

From the repo root:

```bash
go run .
```

Or run a specific file:

```bash
go run 1-goroutines.go
```
